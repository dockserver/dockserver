This is the unified client for [Notifiarr.com](https://notifiarr.com).
The client enables content requests from Media Bot in your Discord Server.
It also provides reports for Plex usage and system health.

## Docker Config File

1. Copy the [config file](https://github.com/Notifiarr/notifiarr/blob/main/examples/notifiarr.conf.example) from this repo.
1. Rename it to `notifiarr.conf`
1. Put it in `/opt/appdata/notifiarr` and set it up to your needs.
1. Restart the container `docker restart notifiarr` and go to [Notifiarr.com](https://notifiarr.com) to add integrations.

For more information please viste [Notifiarr Repo](https://github.com/Notifiarr/notifiarr)
